# C - Variables, if, else, while

This directory contains my solutions for the Holberton School project **"C - Variables, if, else, while"**.

The purpose of this project is to practice:
- declaring and using variables (`char`, `int`, `unsigned int`)
- arithmetic, relational, and logical operators
- `if` and `if...else` conditions
- `while` loops
- printing values and characters using `printf` / `putchar`
- working with ASCII values

## Requirements

- OS: Ubuntu 20.04 LTS
- Compiler: `gcc`
- Compilation flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Style: Betty
- No warnings or errors during compilation
- No use of `system`
- Every file must end with a new line

## Files

- `0-positive_or_negative.c`  
  Prints whether a randomly generated number is positive, negative, or zero.

- `1-last_digit.c`  
  Prints the last digit of a random number and describes it (greater than 5, 0, or less than 6 and not 0).

- `2-print_alphabet.c`  
  Prints the alphabet in lowercase using `putchar`.

- `3-print_alphabets.c`  
  Prints the alphabet in lowercase, then uppercase, using `putchar`.

- `4-print_alphabt.c`  
  Prints the alphabet in lowercase except `q` and `e`.

- `5-print_numbers.c`  
  Prints the digits from 0 to 9.

- `6-print_numberz.c`  
  Prints the digits from 0 to 9 using only `putchar`.

- `7-print_tebahpla.c`  
  Prints the alphabet in reverse (lowercase).

- `8-print_base16.c`  
  Prints all numbers of base 16 in lowercase (0-9 and a-f).

- `9-print_comb.c`  
  Prints all possible combinations of single-digit numbers.

- `100-print_comb3.c`  
  Prints all possible different combinations of two digits (00 to 99, no repeats like 01 and 10).

- `101-print_comb4.c`  
  Prints all possible different combinations of three digits.

- `102-print_comb5.c`  
  Prints all possible combinations of two two-digit numbers (00 01 up to 98 99).

## Compilation

Example:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 filename.c -o output
./output
```
